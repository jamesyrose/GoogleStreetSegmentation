About

This is a UNet w/ Weighted Cross Entropy model used for semnatic segmentation trained off the India Driving dataset. The website produces a semantic segmentation of google street view utilizing Google Cloud Platform. The website is built using JS (JQuery), HTML and CSS.

If this model is to heavy, please use the Flask App to view or view examples here

Use

You may navigate anywhere in the world using the search bar. It fetches the maps using Google Maps Javascript API. Clicking on "Segment Image" will run the model to segment the current view. It will produce an overlay on the current map, which can be turned off an on. Clicking on the "Segmentation" tab will show the last segmented image and its segmentation. Howering over the segmentation images will show you the predict class of that pixel. Segmentation can take some time and may crash your computer depending on its computing power.

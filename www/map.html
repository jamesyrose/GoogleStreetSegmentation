Google Streetview Semantic Segmenation

01

Pure JavaScript

Tensorflow-js version written entirely in HTML, CSS, and JS. Inference is performed on your computer, if you have an old/bad computer, this version may crash on you. Uses light weight model (6 Million Parameters).

Go!

02

Flask App

Flask App using onnx-runtime to perform inference. Built using Python, JS, HTML and CSS. Server resources are limited so this will take some time to load and may crash if over used. Uses light weight model (6 Million Parameters).

Go!

03

Static

Examples of the segmentation using static images, this does not perform inference on demand. Uses full UNet Model (31 Million Parameters)

Go!
